SSH Connection Guide

1 Download the Key Pair

Click the "Download Key" button above to download the private key file for this region.

2 Set Key Permissions

SSH requires the private key to have restricted permissions. Run this command to set the correct permissions:

chmod 600 clusteradmin-us-east-1.pem

3 Connect via SSH

Use the following command to connect to your EC2 instance:

ssh -i clusteradmin-us-east-1.pem ec2-user@0.0.0.0

4 Alternative: Full Path

If you saved the key to your Downloads folder, use the full path:

ssh -i ~/Downloads/clusteradmin-us-east-1.pem ec2-user@0.0.0.0
