## Steps


## Run (manually)

<details>
<summary>App</summary>

```bash
#---------------------------
# Run BE app
#---------------------------

# build
mvn package

# run
java -jar target/springEmployeeSystem-0.0.1-SNAPSHOT.jar
```

```bash
#---------------------------
# Run FE app
#---------------------------

cd springEmployeeSystem/frontend/employee-system-ui

npm run serve
```

</details>


## Run (Docker)

<details>
<summary>App</summary>

```bash
docker-compose up
```

</details>
